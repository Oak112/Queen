# QUEEN Reproduction Package (Copy, No Symlinks)

This folder is a **fully copied** (no symlinks) reproduction package for QUEEN (queen).

Contents:

- `code/queen/`: QUEEN codebase (copied)
- `code/data_adapters/`: dataset adapter scripts (copied)
- `configs/`: configs used for our reproduction
- `scripts/`: helper scripts (dataset prep, video export)
- `docs/`: English step-by-step reproduction guide
- `issues/`: English troubleshooting notes

## Notes

- MiDaS weights (`MiDaS/weights/*.pt`) are **not included** to keep this package smaller.
  The usage guide includes the download command.
